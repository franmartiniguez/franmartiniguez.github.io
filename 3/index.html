Mosaics

For my mosaics, I decided to use the Bilinear interpolation method because while it was a little slower to run the code, on these bigger images, I felt like I could see the difference in quality between Nearest Neighbors and Bilinear. In order to place the two images together, I first had to make a new canvas to put the two images together. I calculated the canvas based on how the Homography matrix changed the one warped image I was going to use. I then calculated how big the actual canvas was going to need to be in order to accomodate the two images. I then overlaid one image onto another. In order to try to reduce image artifacts, I decided to implement an averaging system that determines where there is overlap and then basically takes the average of the two images there.
