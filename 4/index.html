Part 2.2:
For sampling Rays from Images, what it does is first build a uv grid for each image. It then uses the pixel_to_ray function to create rays for every pixel in each image. It then puts them together flattened out and samples randomly from all the available rays in a flattened list. For sample along rays, it takes points evenly along the near and far, except for when there's a perturbation, when instead of sampling at the exact middle where it should ideally be, it adds a small bit of noise to sample randomly. It then returns these points as the samples.
